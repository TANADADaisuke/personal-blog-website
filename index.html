Learning front end web developing

Now I'm really interested in Front-end Web developing and start learning about it!

As the course instructor Daniel says,

Writing code from scratch can feel really difficult. There's nothing scarier than an empty page, whether it's coding or poetry or anything else. But I think there's really something to starting with an empty page and creating those elements out from scratch yourself.

So I'll challenge to make my first web page from scratch.

The goal of this little journey is to create some wonderful web services. Here are the list of what I want to make.

  • Goods storage assistant helps you to store several goods together in storage boxes.
  • English study app helps you to memorize vocabularies and sentences frequently used in daily business.

Look forward to see and use these applications!
